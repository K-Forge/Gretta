INSTRUCCIONES DE EJECUCIÓN
===========================

CONFIGURACIÓN BASE DE DATOS
---------------------------
Antes de iniciar, edita el archivo:
BackEnd/gretta/src/main/resources/application.properties

Comenta o descomenta la sección correspondiente a tu usuario de base de datos (ej. BRIAN, LINA, o con MAYUSCULAS.) o añade tu propia configuración.


POSTGRESQL (Windows)
--------------------
# Iniciar:
net start postgresql-x64-18

# Apagar:
net stop postgresql-x64-18


GUÍA RÁPIDA DE TERMINAL
-----------------------
cd <carpeta>   : Entrar a una carpeta (ej: cd BackEnd)
cd ..          : Regresar a la carpeta anterior
ls             : Listar archivos en la carpeta actual (en Windows usar 'dir')


BACKEND
-------
cd BackEnd/gretta && ./mvnw spring-boot:run


FRONTEND
--------
cd FrontEnd

# Solo la primera vez:
npm install

# Para iniciar:
npm start
